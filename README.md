## Short-Url
> A simple url shortner cli tool made by using ***Python*** for `TERMUX`.That short an url using my own ***Url Shortner api!***

### COMMAND
* `pkg install git`
* `pkg install python`
* git clone https://github.com/fh-rabbi/Short-Url
* cd Short-Url
* `python app.py`
